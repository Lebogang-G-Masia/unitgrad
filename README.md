# Unit Grad

A lightweight C++ autograd engine inspired by Andrej Karpathy's [micrograd](https://github.com/karpathy/micrograd).

This project implements a similar scalar/tensor-based automatic differentiation system in C++, providing a simple and educational look into the mechanics of backpropagation.

## Features
- Tensor-based automatic differentiation.
- Support for basic operations: addition, subtraction, and matrix multiplication.
- ReLU activation function.
- Topological sort-based backpropagation.
